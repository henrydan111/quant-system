## ⚠ 重要 caveat
1. **交易成本按各策略自身保存的默认值**（果仁平台里每个策略各自设定，多为单边 千分之二 或 千分之五，不统一）。做本地复刻对照时，需逐个核对该策略页面/`交易统计` sheet 里的实际成本设置，而非假设统一成本。
2. **16_sm_noc_纯市值正盈利_v4 用单边 千分之三 导出**（非其默认的 千分之二）。原因：果仁导出端对该策略 sid+千分之二 的组合返回了**被污染的服务器缓存**（错误地返回了相邻策略 sm_微盘基准50 的回测结果）。换用未缓存的成本档（千分之三）强制重算后得到该策略**真实**回测（年化 60.00% / 夏普 1.71 / 回撤 35.49%），与默认 千分之二 的差异仅来自略高的成本档，可忽略；其真实 千分之二 结果因平台缓存 bug 暂无法导出。其余 64 个均为各自默认成本下的正确结果。
3. 收益为**总收益口径**（含分红再投资），与本地 EventDriven 总收益可比；与 Vectorized 价格收益不可直接比（见 CLAUDE.md §3.3）。
