# 地缘政治分析框架 (Geopolitical Analysis Framework)

> **用途**：当分析任何地缘政治相关的信息、事件或新闻时，按本框架逐层推进。目标是把"看起来合理的叙事"升级为**可证伪、可校准、可落到资产价格**的结论。
> **版本**：v1.0（2026-06-16）｜本框架是一项常驻分析约定（standing framework）。

## 如何使用（30 秒版）
1. 先做**认识论纪律**检查（第 0 层）——区分事实/推断/推测，给信源评级。
2. 锚定事实（第 1 层）→ 测绘行为体（第 2 层）→ 看结构与历史（第 3 层）。
3. 推因果与情景（第 4–5 层），给**概率 + 触发线**。
4. 落到**影响传导与市场**（第 6 层），分**时间尺度**（第 7 层）。
5. 设**监测与证伪**条件（第 8 层），用**标准模板**输出（第 9 层）。

> 不需要每次九层全做——**重大事件全做**；**日常新闻**只做 0 / 1 / 5 / 6 / 8 的精简版即可。

---

## 第 0 层 · 认识论纪律（先于一切，最重要）

- **三分标注**：每个论断标注 `[事实] / [推断] / [推测]`。
  - 事实 = 可被独立信源证实的已发生之事；推断 = 基于事实的逻辑演绎；推测 = 缺证据的猜想。
- **信源分级 + 交叉验证**：
  - A = 一手官方 / 原始文件；B = 有既往准确记录的专业机构 / 记者；C = 二手转述 / 带立场智库；D = 社媒 / 匿名 / 未证实。
  - 交叉验证 ≥ 2 个**独立**信源后才升级为 `[事实]`。永远问：**谁受益于这个叙事？**（信源的立场与激励）。
- **校准化不确定性，而非对冲词**：禁用"可能、也许、或许、似乎"这类空泛对冲（呼应本项目 CLAUDE.md 规则 #10）。地缘政治本质不确定，但用**「概率 + 置信度 + 证伪条件」**表达，而不是用模糊词、也不是假装确定。
  - ✗ "局势可能升级。"
  - ✓ "未来 3 个月军事升级概率约 30%，置信度中；若观测到 X 则上调至 50%。"
- **基率优先 (base rates)**：先问"这类事件历史上通常如何收场"，再用本次特殊性微调。多数地缘危机以**降级 / 僵局**收场而非全面冲突——别被尾部戏剧性叙事绑架。
- **偏差清单（每次自检）**：
  - 确认偏差——我是否只在找支持既有观点的证据？（用竞争性假设对冲，见第 5 层）
  - 叙事偏差——这个故事是否"太顺"？精彩叙事 ≠ 真相。
  - 镜像偏差 (mirror imaging)——我是否在用**自己的**逻辑假设对手？对手的理性 ≠ 我的理性。
  - 近因 / 可得性偏差——最新最醒目的信息是否被过度加权？
  - 锚定偏差——第一个数字 / 判断是否绑架了后续？
- **留痕纪律**：记录"首次判断"及其时间戳；后续若因新信息修正，**明确标注修正点**（呼应 OOS 神圣——第一次结论要可追溯，事后改口要留痕）。

## 第 1 层 · 事件刻画 (WHAT) — 锚定事实基线

- **5W1H**：何事、何人、何时、何地、为何（声称的）、如何。
- **信号 vs 噪音**：这是一次性事件，还是某个**趋势的节点**？
- **边际信息量**：相对已知，这条新闻**新增了什么**？关键问句——**"是否已被 price in？"** 预期之内的事件市场反应应小；真正的 alpha 在**预期差 (surprise)**。
- **一手核验**：尽量回到原始文本（声明全文、法案条文、制裁清单、官方通稿），而非媒体标题。标题常放大或扭曲。

---

## 第 2 层 · 行为体测绘 (WHO)

对每个主要行为体（国家 / 次国家集团 / 跨国组织 / 关键个人）建卡：

| 维度 | 要问的问题 |
|---|---|
| 核心利益 | 它**真正**想要什么（安全 / 经济 / 政权存续 / 国内政治 / 声望）？ |
| 能力 | 它有多少筹码（军事 / 经济 / 技术 / 外交 / 舆论）？ |
| 约束 | 什么限制它（国内政治、盟友、经济成本、法律、时间窗）？ |
| 意图 | 它打算怎么做？ |
| 红线 | 触发它强硬反应的临界点在哪？ |
| 决策机制 | 谁拍板？是否受选举 / 派系 / 民意约束？ |

**关键原则**：
- **意图 ⊥ 能力**：高能力 + 低意图 ≠ 高威胁；要乘起来看，并**分别监测**。
- **国内政治优先**：多数对外强硬行为是做给国内看的。问"这对它的**国内受众**意味着什么"。
- **博弈结构**：零和还是正和？单次还是重复博弈（重复博弈有声誉约束）？信息对称否？

---

## 第 3 层 · 结构与历史层 (WHY，深层)

- **区分「结构性驱动」与「触发性事件」**：触发事件是火花，结构性张力是火药。没有火药，火花熄灭；**分析要落在火药上**。
- **历史路径依赖**：这对行为体 / 这个议题的历史互动模式是什么？历史不重复，但押韵。
- **三棱镜（同一事件，三种 IR 理论各看一遍，互补不互斥）**：
  - **现实主义（权力 / 安全）**：谁在追求相对权力与安全？力量对比如何变化？安全困境是否在起作用？
  - **自由主义（制度 / 相互依赖）**：经济相互依赖、国际制度、国内利益集团如何约束或塑造行为？相互依赖越深，全面冲突成本越高。
  - **建构主义（认同 / 叙事）**：身份认同、历史记忆、国内叙事、面子如何驱动？很多"非理性"行为在**认同框架**下是理性的。

---

## 第 4 层 · 因果机制与驱动 (HOW)

- **画直接因果链**：A → B → C，每一环标注是 `[事实]` 证据还是 `[推断]`。
- **升级 vs 降级动力**：
  - 推向升级——国内强硬派、沉没成本、安全困境、误判、面子。
  - 推向降级——经济成本、外部调停、**台阶 / off-ramp 的存在**、疲劳。
- **反馈环**：正反馈（报复—反报复螺旋、军备竞赛）会**放大**；负反馈（经济痛感、第三方施压）会**收敛**。识别系统当前由哪种反馈主导。
- **关键不确定变量**：列出 2–4 个最能改变结局的未知量（领导人意志、第三方介入、经济承受力、国内稳定）。

---

## 第 5 层 · 情景分析 (WHAT NEXT)

给出**基准 / 上行 / 下行 + 1 个尾部**情景，每个配：概率、触发条件、演化路径、领先指标。

| 情景 | 概率 | 触发 / 前提 | 关键路径 | 领先指标 (tripwires) |
|---|---|---|---|---|
| 基准（维持现状 / 渐进） | ~% | | | |
| 升级 | ~% | | | |
| 降级 / 缓和 | ~% | | | |
| 尾部（低概率高冲击） | ~% | | | |

- **竞争性假设分析 (ACH)**：列出所有合理假设，用证据去**反驳**而非**证实**——保留**最难被反驳**的那个。这是对抗确认偏差的核心技术。
- **Pre-mortem（事前验尸）**：假设 6 个月后我的主判断被证明**错了**，最可能错在哪一环？把那一环设为重点监测项。
- **概率要校准**：既避免全押"维持现状"（惰性偏差），也避免追逐尾部戏剧性。用基率打底，用本次特殊性微调。

## 第 6 层 · 影响传导 (SO WHAT) — 含市场 / A 股传导矩阵

**分阶**（每一阶都问"然后呢"）：
- **一阶**：直接、即时的后果（制裁生效、航道关闭、价格跳变）。
- **二阶**：行为体的反应与适应（反制、供应链重组、政策对冲）。
- **三阶**：结构性 / 长期重塑（联盟重构、去风险化、技术脱钩、储备货币格局）。

**通用传导渠道**：安全 / 军事、经济 / 贸易、能源与大宗、技术与供应链、金融与资本流动、舆论与认知。

**市场传导矩阵（落到本量化系统 / A 股）**：

| 渠道 | 传导机制 | 受影响资产 / 板块 | 时间尺度 | 与本系统接口 |
|---|---|---|---|---|
| 贸易 / 关税 / 出口管制 | 营收、成本、订单 | 出口链、半导体、设备 | 战术–战略 | 行业因子 / 事件驱动引擎 |
| 资本流动 | 风险偏好 → 外资进出 | 北向重仓白马、核心资产 | 即时–战术 | `north_hold` / `hk_hold` 因子、北向流向 |
| 汇率（人民币） | 估值、外债、定价权 | 航空、进口依赖、出口商 | 即时–战术 | 汇率敏感板块 |
| 大宗商品 | 油 / 气 / 有色 / 农产品 / 稀土 | 能源、化工、有色、农业、军工 | 即时–战略 | 板块轮动、商品 beta |
| 风险偏好 | 避险情绪、波动率 | 全市场 beta、黄金、债 | 即时 | 波动率 / 情绪因子 |
| 政策对冲 | 国家队、产业政策、安全主题 | 中特估、自主可控、军工、粮食安全 | 战术–战略 | 主题策略框架 |
| 板块轮动 | 受益 / 受损再定价 | 自主可控 vs 出口敏感 | 战术 | 行业相对因子 |

对每个市场判断附：**方向、幅度量级、时间尺度、是否已被定价、证伪条件**。
区分**「情绪冲击（数日内均值回复）」**与**「基本面兑现（持续重定价）」**——两者交易方式完全不同。

## 第 7 层 · 时间维度

- **即时（数小时–数日）**：情绪与头寸驱动，常过度反应、易均值回复。
- **战术（数周–数月）**：政策反应与基本面开始兑现。
- **战略（数季–数年）**：结构性重塑（脱钩、联盟、产业链迁移）。

**关键**：不同时间尺度的结论**可以相反**（短期避险抛售 vs 长期受益于国产替代）。**必须分层给结论，不要混为一谈。**

---

## 第 8 层 · 监测与证伪

- **领先指标清单**：列出 3–6 个**可观测、可验证**的指标，作为情景切换的触发线 (tripwires)。
- **证伪条件**：明确写出"若观测到 X，则当前主判断**作废 / 反转**"。**没有证伪条件的判断不是分析，是信仰。**
- **复盘触发器**：设定何时（时间或事件）强制重评。
- **校准复盘**：事后对照实际结果，回看概率给得是否过高 / 过低，迭代下次校准。

---

## 第 9 层 · 标准化输出模板

> **结论先行 (BLUF, Bottom Line Up Front)**，再展开。

```
【一句话结论 / BLUF】<方向 + 概率 + 时间尺度 + 最关键的一个证伪条件>

【事实基线】   [事实]…（信源等级 A/B/C/D）
【关键不确定性】最能改变结局的 2–3 个未知量
【行为体与动机】各方真正想要什么 / 红线
【情景与概率】 基准 X% / 升级 Y% / 降级 Z%（+ 触发线）
【市场传导】   渠道 → 资产 → 方向 / 量级 / 时间尺度 / 是否已定价
【时间分层】   即时 / 战术 / 战略 分别的判断
【监测与证伪】 领先指标 + 推翻条件 + 复盘时点
【认识论标注】 哪些是推测、信源最弱的环节在哪
```

## 快速自检清单

- [ ] 每个论断都标了 事实 / 推断 / 推测？
- [ ] 信源做了分级与交叉验证？立场与激励看了？
- [ ] 用了**概率 + 置信度 + 证伪**，而非模糊对冲词？
- [ ] 对了**基率**？没有被尾部戏剧性绑架？
- [ ] 做了竞争性假设 / pre-mortem 对抗确认偏差？
- [ ] 区分了**意图与能力**？
- [ ] 落到了具体资产 / 板块，并标了**是否已被 price in**？
- [ ] 分了**时间尺度**？
- [ ] 写了"**什么会让我改变结论**"？
