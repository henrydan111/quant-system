## 0. 总览

65 个 slct 策略共用到：
- **194 个命名指标**：99 自定义（你的 452 导出库）、14 内置（文档财务表有公式）、81 内置/行情/事件/朝阳永续（prose 或 AI 确认）。
- **85 条去重内联 `公式()`**（多为路径/动量类）。
- **3 条大盘择时公式**。
- **43 个函数**。

指标体系本质是一套 **GARP + 质量 + 价量 + 事件** 的多因子库，绝大多数自定义指标是在果仁内置财报字段上做「单季化 / TTM化 / 同比环比 / 中性化」的二次加工。

## 1. 函数速查（策略实际用到的 43 个）

### 1.1 时序窗口函数（对单只股票在过去 N 个**交易日**聚合，跳过停牌日；N=0 表示自上市起）
| 函数 | 语义 |
|---|---|
| `Ref(指标,N)` | N 个交易日前的值（**含停牌日**计数）。N=0 取上市首日（2007前股票取 2007-01-04）。 |
| `MA(指标,N)` | 过去 N 日简单均值。 |
| `WMA(指标,权重,N)` | 过去 N 日按「权重指标」加权的均值（如成交量加权均价≈成本线）。 |
| `Sum(指标,N)` | 过去 N 日求和。 |
| `Max/Min(指标,N)` | 过去 N 日最大/最小值。 |
| `Stdev(指标,N)` | 过去 N 日标准差（波动率常用 `Stdev(1日涨幅,N)*sqrt(N)`）。 |
| `Skewness(指标,N)` | 过去 N 日偏度。 |
| `Rank(指标,N,顺序)` | 当前值在过去 N 日时序中的名次（顺序0=由小到大）。**时序**排名，非截面。 |
| `Corr(指标1,指标2,N)` | 过去 N 日两指标的时序相关系数。 |
| `SlopeXY(Y,X,N)` | 过去 N 日 Y~X 时序线性回归斜率。 |

### 1.2 截面函数（同一交易日、横截面；范围 0=全市场,1=一级行业,2=二级行业）
| 函数 | 语义 |
|---|---|
| `HAvg/HStdev/HMed(指标,范围)` | 截面均值/标准差/中位数。 |
| `HRank(指标,顺序,范围)` | 截面名次。 |
| `HPercentRank(指标,顺序,范围)` | 截面分位（0~1）。顺序1=由大到小（值越大分越高）。 |
| `HWinsorize(指标,上分位,下分位,范围)` | 截面分位去极值。 |
| `HNeutralize(Y,X,范围)` | **截面**回归 Y~(行业哑变量 + X)，返回**残差**（剔除行业与 X 的影响）。果仁中性化核心函数。 |
| `HNeutralizeMI(指标[,默认=总市值])` | 一步做「市值 + 行业」中性化（MI=Market+Industry）。 |
| `SSlopeXY(Y,X,股票池,范围)` | 指定股票池内截面 Y~X 回归斜率。 |
| `SAvg/SMed(指标,股票池,范围)` | 指定股票池内截面均值/中位数。 |

### 1.3 财报季度函数（季报指标的纵向处理）
| 函数 | 语义 |
|---|---|
| `TTM(季报指标,前移季数)` | 最新 4 个季度之和。`TTM(x,0)`=最近 12 个月；`TTM(x,4)`=1 年前的 TTM。 |
| `AccuQ(季报指标,前移年数)` | 年内累计（YTD）。`AccuQ(x,0)`=本年至今；`AccuQ(x,1)`=去年同期 YTD。 |
| `Annual(季报指标,前移年数)` | 年报值。`Annual(x,0)`=最近年报。 |
| `RefQ(季报指标,N)` | N 个季度前的值。`RefQ(净利润(单季),4)`=去年同季单季净利。 |
| `AvgQ(季报指标,季数,前移)` | 最近若干季度均值。 |
| `StdevQ(季报指标,季数)` | 最近若干季度标准差（盈利稳定性）。 |
| `SumQ(季报指标,季数)` | 最近若干季度求和。 |

### 1.4 数学/逻辑/其它
| 函数 | 语义 |
|---|---|
| `If(cond,a,b)` / `And/Or` / `Greater/Less(...)` | 条件、逻辑与或、取多值中的最大/最小。 |
| `IfNULL(x,y)` | x 为空则取 y（缺失值兜底，财报类常用）。 |
| `abs / ln / log / square / max / min / mod` | 标准数学。 |
| `Stdevm(x1..x8)` | 多个指标（最多8）的标准差（如4个季度单季营收的离散度）。 |
| `TickerValue(指标,代码)` | 取指定股票代码的指标值（AH 溢价、跨标的比较用）。 |
| `KLast(指标,K线级别)` | 高级别 K 线最后值（如 `KLast(收盘价,m)`=月线收盘）。 |
| `Timing(指标,下限,上限)` | **择时开关**：指标<下限→1，>上限→0，否则保持昨值（迟滞）。大盘择时核心。 |

## 2. 内置指标定义

### 2.1 行情/技术（来自 [果仁选股指标详解](../../../../Knowledge/果仁帮助文档/果仁选股指标详解.txt) prose）
- **总市值 / 流通市值**：A 股当日总/流通市值。
- **收盘价 / 后复权收盘价**：后复权计入分红拆股，用于跨期收益/动量。
- **N日平均成交额/成交量**：过去 N 交易日均值，停牌日不计；非停牌日<N 时为空。
- **N日涨幅**：过去 N 日累计涨跌幅（**含**停牌日计数）。
- **N日换手率**：过去 N 日累计换手（成交股数/总股本）。
- **N日乖离率**：`后复权收盘价 / N日复权均价 - 1`。
- **N日波动率**：`Stdev(1日涨幅,N) * sqrt(N)`。
- **历史贝塔**：过去 250 日个股对沪深300 的回归贝塔。
- **RSI**：14 日相对强弱（`SMA(greater(1日涨幅,0)*因子,14,1)/SMA(abs(1日涨幅)*因子,14,1)*100`）。
- **股价振幅**：`(最高价-最低价)/前日收盘价`。
- **上市天数 / 交易天数**：自然日 / 交易日计。
- **股东数60日变化率 / 机构持股比例 / 机构持股环比增长 / 户均持股数**：见 prose 文档。
- **新增流通股数/占比（当日/未来20日/未来60日）**：`新增流通股数/流通股本`，反映解禁/增发稀释（前瞻事件）。

### 2.2 财务（来自 [财务指标公式](../../../../Knowledge/果仁帮助文档/财务指标公式.txt) 表 + backbone §B）
策略直接用到且文档有公式的 14 个，公式见 backbone §B。要点：
- **EP=TTM(归母净利,0)/总市值**、**BP=归母权益/总市值**、**SP=1/PS**：估值倒数（越大越便宜）。
- **负债资产率=负债合计/资产总计**；**净资产收益率(ROE)=TTM(净利,0)/AvgQ(权益,4,1)**。
- **净利润增长/营业收入增长**：`(TTM(x,0)-TTM(x,4))/abs(TTM(x,4))`（TTM 同比）。
- **每股收益/每股净资产/每股股息**：除以总股本。

### 2.3 朝阳永续/评级/事件/中性化（帮助文档未收录 → 见 [guorn_aichat_indicator_defs.md](guorn_aichat_indicator_defs.md)）
关键结论：
- **中性EP/BP/SP/波动率/换手率** = `HNeutralize(原指标, log(总市值), 行业)` 取残差（剔除行业+规模）。⚠AI 推断，用前实测。
- **真实负债资产率** = `负债合计/(资产总计-商誉-无形资产-开发支出)`（剔除虚资产的真实杠杆）。
- **ILLIQ** = Amihud `Sum(|1日涨幅|/成交额(亿),N)/N`，默认 N=5。
- **预期\*族** = 多机构分析师一致预期（FY1/FY2），如 `预期EP第2年=预期净利FY2/总市值`。
- **评级\*族** = 过去6个月研报统计：调高家数、增持家数、机构数、平均评级分(1-5)、净调高比例。
- **果仁风险预警25版** = 退市/重大风险综合标记（审计意见异常/年报逾期/国九条退市/净资产为负/被预警）。

## 3. 自定义指标分析（按族；精确公式 + 依赖展开见 backbone §A）

> 命名约定：`Q`=单季、`TTM`=滚动12月、`A`=年度、`Gr%PY`=同比增长率(vs 去年同期 TTM)、
> `%PYQ`=对去年同季单季、`DiffPQ`=环比差(vs 上季)、`%`某分母=比率。

### 3.1 成长/同比环比族（策略最核心，出现频率最高）
计算模式统一为 `(本期 - 去年同期)/|去年同期|`，分子分母用 `RefQ(...,4)`（季度同比）或 TTM。
- **CoreProfitQGr%PY**（用23次/22策略）：核心利润单季同比。核心利润 `CoreProfitQ = 营业收入-营业成本(单季)-(管理+销售+财务费用单季)-营业税金及附加(单季)`，剔除投资收益/营业外等非经常项，比净利润更反映主业。
- **EpsExclXorQGr%PY**：扣非每股收益单季同比（`EpsExclXorQ = IfNULL(扣非净利单季, 归母净利单季)/总股本`）。
- **ROETTMDiffPQ**：`净资产收益率 - RefQ(净资产收益率,1)`，ROE 的**环比改善**（趋势，非水平）。
- **业绩快报归母净利QGr%PY / 业绩预告净利润QGr%PYQ_v1**：用快报/预告数据抢先一个季度捕捉单季同比（含 `accuQ` 累计→单季还原）。
- **所得税费用QGr%PY / SalesQGr%PY / 营业收入TTM同比** 等：同模式套在不同财报字段上。
- 分析：这族是「**加速度**」因子——不看盈利水平而看**同比/环比变化**，捕捉景气拐点。注意分母用 `abs()` 防负利润翻转符号，但当去年同期≈0 时仍会爆量（需配合去极值/中性化模板，见 3.6）。

### 3.2 核心利润 / 盈利质量族
- **CoreProfitQ / CoreProfitQ_v1 / 预期CoreProfitA**：核心利润的单季/预测口径，是 3.1 多个增长指标的基底。
- **GrossProfit%AssetsQ**：`(营业收入-营业成本)(单季)/资产总计(单季)`，毛利/总资产 = 新版「质量」因子（Novy-Marx 毛利率资产比的单季版）。
- **EBITDAQ / EBITDAQ%EV**：`EBITDA单季 / 企业价值`，剔除资本结构的估值。
- 分析：盈利质量族关注「真实赚钱能力」，规避会计利润被投资收益/补贴粉饰。

### 3.3 自由现金流族
- **FCFQ_重算**：`经营现金流净额(单季)-购建固定资产等(单季)+处置固定资产收回(单季)+折旧摊销(单季)`，自算 FCF。
- **NetFCFTTM**：`TTM(FCFQ_重算)-分红总金额`，分红后可再投资的 FCF。
- **FCFQ%总市值 / FCFQ_重算Gr%PYQ**：FCF 收益率、FCF 单季同比。
- 分析：现金流因子比利润更难操纵，价值策略（value_FCF_*）核心。

### 3.4 预期/业绩预告族（抢先信息）
- **业绩预告净利润Q / 业绩预告净利润Q_v1**：`(预计净利上限+下限)/2 - accuQ(净利单季,0)`，用预告中值还原**单季**预告净利。
- **预期扣非净利润A_by_CV / _by_PCT / 预期CoreProfitA**：用历史 accuQ 占全年比例 + 预告/一致预期，外推全年净利。
- 分析：A 股财报披露慢，预告/快报提前 1 季度释放信息，这族在「净利润断层 / 业绩超预期」策略里抢跑。⚠ 对本地系统是 **PIT 高危区**：必须按预告**公告日**对齐，严禁用 end_date。

### 3.5 估值族
- **EPCoreProfitQ_v1**（核心利润 EP）、**BP筹资市值比调整**、**EBITDAQ%EV**、各种 `X/总市值`。
- **BP筹资市值比调整** = `HNeutralize(hpercentrank(BP,1,0), hpercentrank(SumQ(筹资现金流入,4)/总市值,1,0), 0)`：把 BP 分位对「融资强度」分位做中性化——剔除「靠融资撑起的高账面价值」。设计精巧。

### 3.6 中性化/标准化模板族（可复用模板，带 `{0}` 参数）
- **指标1指标2中性化** = `HNeutralize({0}, ln({1}), {2})`：通用「对数因子」中性化模板。
- **指标标准化后中性化MI** = `HNeutralizeMI(HWinsorize({0},0.05,0.05,0))`：先 5% 去极值再市值+行业中性化——**因子预处理标准流水线**。
- 分析：这族不是 alpha 本身，而是**工具**，把任意原始因子做成「干净的」截面残差。对本地系统极有参考价值（对应我们的 `cs_zscore`/`winsorize`/中性化）。

### 3.7 流动性/动量/路径族
- **股价振幅%当日成交额10日** = `MA(股价振幅/当日成交额,10)`：振幅/成交额≈类 Amihud 非流动性。
- **成交量_近3周放量 / 成交量_连续三周缩量**：量能形态。
- 路径/动量类多数以**内联公式**出现（见 §4）。

### 3.8 过滤/风控模板族（筛选 & 卖出条件用）
- **基础过滤_退市风险_v1(收盘价)** = `OR(收盘价<{0}, sum(预期ST2021,30)>0, sum(果仁风险预警25版,30)>0, 过去N日重大违规数量(30)>0)`：低价+ST+风险预警+违规的综合排雷开关（=1 触发）。几乎每个股票策略都挂。
- **基础过滤_流动性\* / 退市风险_新增** 等：成交额门槛、风险新发生事件。
- 分析：这族是**第2层（universe/eligibility）**逻辑，不是 alpha；对应我们四层管线的 Layer-2 mask。

> 其余自定义指标（壳价值、RIM、EV、有息债务、上季发布的预计…等）均为上述族的变体或更细的财报重算，逐条公式见 backbone §A，依赖链已展开。

## 4. 内联公式 & 大盘择时

### 4.1 高频内联公式（路径调整动量，反复出现于成长/动量策略）
形如：
```
SUM(IF(当日涨停标记=1,0,LOG(后复权开盘价/REF(后复权收盘价,1))),250)
 - SUM(IF(当日涨停标记=1,0,LOG(后复权开盘价/REF(后复权收盘价,1))),20)
```
拆解：
- `LOG(后复权开盘价/REF(后复权收盘价,1))` = 隔夜对数收益（今开/昨收）。
- `IF(当日涨停标记=1,0,…)` = **剔除涨停日**（涨停买不进，去掉其贡献，防失真）。
- `SUM(…,250) - SUM(…,20)` = 250 日累计 − 20 日累计 = **(250−20)=过去230日、剔除最近20日**的中长期动量（经典「动量去掉最近一月反转」）。
- 与本地系统 `mom_route_*` / 隔夜动量因子（记忆 [overnight momentum](../../../../) ）高度同源——值得直接对标。

其余 85 条去重内联公式（业绩预告还原、行业内排名、量价比等）逐条见 backbone §C。

### 4.2 大盘择时公式（3 条，均用 `Timing` 迟滞开关）
- `If(OR(ma(指数收盘(111001),5)>ma(指数收盘(111001),200), Timing(ma5/ma200,0.85,0.95)),1,0)`：5 日均线 > 200 日均线（趋势在上）**或** 均线比在 0.85–0.95 迟滞区 → 满仓(1)，否则空仓(0)。`111001` = 某基准指数。
- 另两条加了 `1-指数收盘/Max(指数收盘,60)<0.1`（距 60 日高点回撤<10%）与回撤 Timing，做**双择时**。
- 分析：这是叠加在选股上的**仓位开关**（0/1），对应我们的大盘择时层。⚠ 本地复刻需 leverage≤1（满仓=1×，不可放大）。

## 5. 对接本地量化系统的建议

| 果仁概念 | 本地对应 | 备注 |
|---|---|---|
| 单季/TTM/同比环比族 (Gr%PY/%PYQ/DiffPQ) | PIT 财报增长字段 `pit_*_yoy` / `derive_single_quarter_value` | 我们已有单季还原与 YoY；可直接复刻 CoreProfitQGr%PY 一类 |
| 业绩预告/快报抢跑族 | `forecast` 数据集 + 预告公告日 PIT | **PIT 高危**：按 ann_date 对齐，禁用 end_date |
| 中性化/标准化模板 (HNeutralize/HNeutralizeMI/HWinsorize) | `cs_zscore` / `winsorize` / 市值行业中性化 | 语义一致，可 1:1 映射 |
| 路径动量(剔除涨停的隔夜动量) | `mom_route_*`、隔夜动量因子 | 高度同源，优先对标验证 |
| 基础过滤_退市风险/流动性 | Layer-2 universe mask（ST/退市/流动性） | 对应四层管线 Layer-2，非 alpha |
| 大盘择时 Timing 开关 | 择时层，仓位 0/1 | unlevered，1× 上限 |
| 朝阳永续 预期\*/评级\* 族 | 我们的 `report_rc`(分析师预期) | 数据源相近；eps_diffusion 已是此类 |

**落地优先级**：① 中性化模板（工具，立即可用）→ ② 成长同比环比族（与现有 PIT 增长字段对齐）→ ③ 路径动量（与隔夜动量对标）→ ④ 预期/评级族（report_rc 已部分覆盖）。所有复刻须走 ledger+Qlib，禁止手搓 PIT。
