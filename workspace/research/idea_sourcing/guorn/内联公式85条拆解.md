# 果仁 slct 策略 — 85 条内联公式逐条拆解

> 内联公式 = 策略「排名/筛选条件」里以 `公式(...)` 直接写的表达式（非命名指标）。共 149 处、去重 **85 条**。
> 编号 [1]–[85] 对应 [inline_formulas_classified.json](inline_formulas_classified.json)（按用量降序）。
> 函数语义见 [指标拆解与分析.md §1](指标拆解与分析.md)；自定义指标公式见 [indicator_reference_auto.md](indicator_reference_auto.md)。
>
> **约定**：`%(a,b)` = 果仁百分比变化函数 = `a/b − 1`（文档未收录，按上下文判定，用前实测）；
> `RefQ(x,4)` = 去年同季；`TTM(x,0)` = 最近12个月；`HAvg(x,1)` = 个股**一级行业内**当日均值；
> 「单季」= 当季单季值，「单季环比增长」= 该字段的环比增长内置变体。

## A. 价量 / 动量 / 路径类（20 条）

**[6]** `SUM(IF(当日涨停标记=1,0,LOG(后复权开盘价/REF(后复权收盘价,1))),250) − SUM(...,20)`
- `LOG(后复权开盘价/REF(后复权收盘价,1))` = 隔夜对数收益（今开/昨收）；`IF(当日涨停标记=1,0,…)` = **涨停日置 0**（涨停买不进，剔除其贡献）。
- `SUM(·,250) − SUM(·,20)` = 过去 250 日累计 − 过去 20 日累计 = **过去 230 日、剔除最近 1 个月** 的隔夜路径动量。经典「动量去掉最近一月反转」。

**[8]** 同 [6]，窗口 `120 − 20` = 过去 100 日（剔除最近月）隔夜路径动量（中期版）。

**[62]** `SUM(IF(当日涨停标记=1,0,LOG(后复权开盘价/REF(后复权收盘价,1))),120)` = [8] 的被减项单独使用 = 过去 120 日剔除涨停的隔夜路径动量（**不**剔除最近月）。

**[61]** `SUM(IF(当日涨停标记=1,0,LOG(1日涨幅+1)),250) − SUM(...,20)` = 与 [6] 同构，但用 **全日对数收益** `LOG(1日涨幅+1)`（含日内）而非隔夜收益。过去 230 日（剔最近月）全日路径动量。

**[16]** `SUM(1月涨幅,36) − 1月涨幅` = 过去 36 个月累计月涨幅 − 当月 = **剔除当月的 35 个月长期动量**（≈3 年动量去最近月）。

**[60]** `SUM(1月涨幅,36)` = 过去 36 个月累计动量（含当月）= [16] 的被减项。

**[17]** `SUM(IF(1月涨幅>0,1,0),13) − IF(1月涨幅>0,1,0)` = 过去 13 个月「上涨月计数」− 当月 = **剔除当月的 12 个月上涨月数**（动量的「胜率/一致性」口径，0–12）。

**[49]** `MA(后复权收盘价,3)/MA(后复权收盘价,200)` = 3 日均价 / 200 日均价 = **短长均线比**（趋势强度，>1 多头）。

**[29]** `20日波动率/250日波动率` = 短期波动 / 长期波动 = **波动率放大/收敛比**（>1 近期变躁动）。

**[53]** `N日平均成交量(10)/250日平均成交量` = 10 日均量 / 250 日均量 = **放量比**（量能扩张）。

**[56]** `STDEVM(N日换手率(1),(5),(10),(20),(60),(120))` = 6 个不同窗口换手率的离散度 = **换手率期限结构的不稳定性**（换手在不同周期是否一致）。

**[69]** `corr(后复权收盘价,当日成交量,20)` = 过去 20 日 **价量相关系数**（量价齐升 vs 背离）。

**[63]** `SUM(后复权收盘价/MAX(后复权收盘价,250)>0.9,5)` = 最近 5 日中「价格≥250 日最高价 90%」的天数（0–5）= **贴近年内新高的持续度**。

**[59]** `SUM(1+1日涨幅 − 历史贝塔*指数涨幅(899050),20)` = 每日 `1 + 个股收益 − β×指数(899050)收益` 累计 20 日 ≈ 20 + **20 日累计特异（剔除 β·大盘）收益** = 残差动量/选股 alpha 动量。`899050`=指定基准指数代码。

**[13]** `动量_风险调整(60)/动量_风险调整(250)` = 风险调整动量（自定义）的 **短/长比** = 动量加速度。

**[74]** `动量_风险调整(20)/动量_风险调整(120)` = 同 [13]，更短窗（20/120）。

**[50]** `MA(涨停家数比例,60)` = 60 日均「全市场涨停家数占比」= **市场情绪/赚钱效应**（大盘择时输入，全市场同值）。

**[51]** `MA(跌停家数比例,60)` = 60 日均跌停家数占比 = 市场恐慌度。

**[52]** `MA(跌停家数比例,60) − MA(跌停家数比例,250)` = 跌停占比 60 日均 − 250 日均 = **恐慌情绪的边际变化**（筛选/择时）。

**[48]** `MA(加权PEXPB5年分位点,5) > MA(加权PEXPB5年分位点,120)` = (PE×PB 的 5 年分位) 的 5 日均 > 120 日均 → 返回 1/0 = **估值分位上行**择时开关。

## B. 成长加速度类（短期增长 − 长期增长，7 条）

**[3]** `营业收入(单季环比增长) − 营业收入增长` = 单季营收**环比**增长 − 营收 TTM **同比**增长 = 近端增速相对中枢的加速度。

**[4]** `COREPROFITQGr%PQ − COREPROFITTTMGr%PY` = 核心利润单季**环比** − 核心利润 TTM **同比** = 核心利润的增速拐点。

**[5]** `COREPROFITTTMGr%PY − COREPROFITTTMGr%3Y` = 核心利润 TTM 同比 − TTM 3 年复合 = **近一年增速相对 3 年中枢的加速**。

**[7]** `营业收入增长 − 营业收入3年复合增长` = 营收 TTM 同比 − 3 年复合 = 营收加速度（内置版）。

**[84]** `预期盈利2年复合增长 − 净利润3年复合增长` = 未来 2 年**预期**盈利增速 − 过去 3 年**实际**净利增速 = 「预期相对历史」的加速（前瞻景气）。

**[85]** `预期营收2年复合增长 − 营业收入3年复合增长` = 同 [84] 的营收版。

**[27]** `(营业收入(单季) − refq(营业收入(单季),4))/总市值` = 单季营收同比增量 / 市值 = **营收同比增长的「市值标准化」额**（规模可比的营收超额）。

## C. 盈利质量：趋势 & 稳定性（12 条）

**[18] / [82]** `销售毛利率Q − RefQ(销售毛利率Q,4)` = 单季毛利率**同比改善**（毛利率趋势，去季节性）。([82] 为同式小写写法)

**[83]** `销售毛利率Q − 销售毛利率` = 单季毛利率 − TTM 毛利率 = 近端毛利率相对中枢的偏离（边际改善）。

**[14]** `HAVG(销售毛利率Q − RefQ(销售毛利率Q,4),1)` = [18] 的**一级行业内均值** = 行业层面的毛利率改善景气。

**[54]** `REFQ(净资产收益率,0) − REFQ(净资产收益率,1)` = ROE 当季 − 上季 = **ROE 环比改善**（等价于命名指标 `ROETTMDiffPQ` 的 REFQ 写法）。

**[31]** `COREPROFITQGR%PY / GREATER(STDEVQ(COREPROFITQGR%PY,4),0.001)` = 核心利润同比 / 其近 4 季标准差（下限 0.001 防除零）= **增长的信息比率**（高增长且稳定者得分高）。

**[71]** `stdevQ(CoreprofitQGr%PY,12)` = 核心利润同比 12 季标准差 = **增长波动率**（越低越稳，常作反向/质量项）。

**[57] / [72]** `STDEVQ(ROECOREQ,12)` = 核心 ROE（自定义 `ROECoreQ`）12 季标准差 = **盈利能力稳定性**。

**[58]** `STDEVQ(SALESQGR%PY,12)` = 营收单季同比 12 季标准差 = 营收增长稳定性。

**[68]** `avgq(销售毛利率Q,4)/stdevq(销售毛利率Q,4)` = 近 4 季毛利率均值 / 标准差 = 毛利率的**信息比率**（高且稳）。

**[73]** `stdevq(销售毛利率Q,8)` = 近 8 季毛利率标准差 = 毛利率稳定性（中期）。

**[79]** `管理费用/营业利润` = 管理费用占营业利润比 = **费用效率/管理质量**（越低越好）。

## D. 业绩预告 / 一致预期 — 超预期 & 修正（10 条）

**[9]** `(净利润 − 预期净利润Q)/ABS(预期净利润Q)` = (实际单季净利 − 预期单季净利)/|预期| = **盈利超预期幅度 SUE**（净利润断层核心）。

**[23]** `(业绩预告净利润Q − 预期净利润Q)/ABS(预期净利润Q)` = 用**预告**值替代实际的超预期幅度（抢先 1 季度）。

**[24]** `(业绩预告净利润Q − 预期净利润Q)/总市值` = 同 [23]，分母换**市值**（规模标准化的超预期额）。

**[25]** `(业绩预告营业收入Q − 预期营业收入Q)/ABS(预期营业收入Q)` = 营收版超预期幅度。

**[26]** `(业绩预告营业收入Q − 预期营业收入Q)/总市值` = 营收超预期额 / 市值。

**[11]** `业绩预告净利润QGr%PYQ_v1` = 单条命名自定义指标包装（预告单季净利对去年同季的增长，公式见 backbone）。

**[19]** `%(if(财报发布天数=1,后复权开盘价,ref(后复权开盘价,财报发布天数−1)), REF(后复权收盘价,财报发布天数))`
- 分子 = **财报发布当日的复权开盘价**（今天就是发布日则取今开，否则取「财报发布天数−1」日前的开盘 = 发布日开盘）；分母 = 发布日**前一日收盘**。
- `%(开,昨收)` = **财报披露当日的隔夜跳空收益** = 净利润断层/PEAD 的事件信号。

**[12]** `%(评级机构数, REF(评级机构数,60))` = 评级机构数相对 60 日前的**变化率** = 卖方覆盖度的增长（关注度上升）。

**[20]** `%(预期净利润1年, REF(预期净利润1年,60))` = FY1 预期净利相对 60 日前 = **盈利预期 60 日上修幅度**（分析师上调）。

**[21]** `%(预期盈利2年复合增长, ref(预期盈利2年复合增长,60))` = 2 年预期增速的 60 日修正。

## E. 估值 / 股息 / 现金流（16 条）

**[1]** `(营业收入(单季) − 营业成本(单季))/EV` = 单季毛利 / 企业价值 = **毛利-EV 收益率**（剔除资本结构的价值）。

**[2]** `营业收入(单季)/总市值` = 单季营收 / 市值 = **SP（市销率倒数）单季版**。

**[66]** `TTM(所得税费用,0)/总市值` = 所得税 TTM / 市值 = **税额收益率**（税是较难粉饰的真实盈利代理）。

**[76]** `总市值/(预期净利润1年*0.5 + 预期净利润第2年*0.5)` = 市值 / (FY1、FY2 预期净利各半) = **前瞻混合 PE**（值越小越便宜）。

**[64]** `TTM(COREPROFITQ,4)*DIV%NETINC/总股本/收盘价` = 上年核心利润 TTM × 派息率(`DIV%NETINC`) / 总股本 / 收盘价 = **每股预期分红 / 价格 ≈ 预期股息率**（用核心利润×派息率近似分红）。

**[77]** `研发费用(TTM)/总市值` = 研发 TTM / 市值 = **市研率倒数**（研发投入估值）。

**[78]** `研发费用(单季度)/营业收入(TTM)` = 单季研发 / TTM 营收 ≈ 研发强度（口径混用单季/ TTM，偏瞬时）。

**[22]** `(TTM(经营现金流量净额,0) − TTM(营业收入,0))/资产总计` = (CFO − 营收) / 总资产。注：量纲上 CFO 与营收相减不常规，更像**现金流相对营收缺口 / 资产**的自定义质量项（用前核对作者意图）。

**[80]** `经营现金流量净额(TTM)/营业收入(TTM)` = **现金流销售比**（净现比，盈利含金量）。

**[81]** `股息率TTM − 10年国债收益率` = 股息率 − 无风险利率 = **股债利差/性价比**（FED model 式，越高越有吸引力）。

**[28]** `10年国债收益率 − SMED(股息率TTM, STRG_价值红利100)` = 国债收益率 − 「价值红利100」池**中位股息率** = [81] 的反向（在 0 附近做红利择时阈值）。

**[70]** `ma(SAVG(股息率TTM, STRG_价值红利100, 0),20)` = 「价值红利100」池平均股息率的 20 日均 = 红利池**整体股息率水平**（择时/参照）。

**[65]** `TTM(FCFQ_重算,4)` = 自由现金流（自定义 `FCFQ_重算`）的 **1 年前 TTM**（用于同比/历史对照）。

**[32]** `FCFTTM_重算GR%PY` = 自由现金流 TTM 同比（命名自定义指标包装，公式见 backbone）。

**[67]** `annual(经营现金流量净额,0)+…+annual(…,4)` = 近 **5 个年报**经营现金流之和 = 长期现金创造力（筛选门槛）。

**[75]** `市净率` = 直接用内置 PB（筛选门槛）。

## F. 中性化 / 截面（12 条）

**[15]** `HNEUTRALIZE(SALESQGR%PY, LN(总市值))` = 营收单季同比对 ln(市值)+行业回归取**残差** = 市值/行业中性的营收增长。

**[41]** `HNEUTRALIZE(CoreProfitQGr%PY, LN(总市值))` = 核心利润同比的市值/行业中性化。

**[42]** `HNeutralize(评级机构数, ln(总市值), 0)` = 覆盖机构数对 ln(市值) 中性（范围 0=全市场，仅市值不含行业）= 剔除「大票天然多覆盖」后的超额关注度。

**[43]** `Hrank(评级机构数,1,0)` = 评级机构数全市场**截面排名**（顺序 1=由大到小，覆盖越多名次越靠前）。

**[33]** `HAVG(CoreProfitQGr%PY,1)` = 核心利润同比的**一级行业内均值** = 行业景气（同行业个股同值）。

**[35]** `HAVG(ROETTMDiffPQ,1)` = ROE 环比改善的行业内均值 = 行业 ROE 趋势。

**[36]** `HAVG(SalesQGr%PY,1)` = 营收单季同比的行业内均值 = 行业营收景气。

**[34]** `HAVG(OPCFNPDIFF%NP,1)` = 自定义「经营现金流-净利差 / 净利」(`OPCFNPDIFF%NP`，盈利含金量) 的行业内均值。

**[37]** `HAVG(中性ROE,1)` = 中性 ROE 的行业内均值。

**[38] / [39]** `HAVG(预期盈利2年复合增长, 1 / 2)` = 预期盈利增速的**一级 / 二级行业**内均值 = 行业前瞻景气（[38] 一级、[39] 二级）。

**[40]** `HAVG(预期营收2年复合增长,1)` = 预期营收增速的一级行业内均值。

## 小结

- **真正的 alpha 表达式**集中在 A（路径动量/价量）、B（成长加速度）、C（盈利质量趋势与稳定性）、D（超预期与预期修正）四族；E/F 是估值与因子预处理；G 是筹码；H 是 ETF 容器。
- **复刻优先级**（与本地系统对接）：
  1. [6][8][61][62] 剔除涨停的隔夜/全日路径动量 → 对标本地隔夜动量 / `mom_route_*`；
  2. [9][19][23]-[26] 超预期/财报跳空（SUE、净利润断层）→ 走 `forecast`/`report_rc` 的 PIT 事件流；
  3. [31][57][68][71]-[73] 增长/ROE 的信息比率与稳定性 → 我们尚缺，易加；
  4. [15][41][42] 市值/行业中性化 → 对标 `cs_zscore`+中性化模板。
- **用前必校**：`%(a,b)` 函数定义、[22] 的 CFO−营收量纲、`动量_风险调整`/`OPCFNPDIFF%NP`/`DIV%NETINC` 等自定义子指标的精确公式（见 backbone §A）、以及 H 组各 ETF 代码与基金名的对应。
