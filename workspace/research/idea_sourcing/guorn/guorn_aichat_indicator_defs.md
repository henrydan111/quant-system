## A. 中性化族 (★AI 推断 — 平台未给原文，按知识库模式推断)

果仁默认中性化 = `HNeutralize(目标, log(总市值), 行业范围)`，OLS 线性回归取**残差**，行业用申万一级(范围=1)哑变量，市值用 `log(总市值)`。

| 指标 | 中性化口径 (AI 推断) |
|---|---|
| 中性EP | `hNeutralize(EP, log(总市值), 1)` — 行业+对数市值，残差 |
| 中性BP | `hNeutralize(BP, log(总市值), 1)` |
| 中性SP | `hNeutralize(SP, log(总市值), 1)` |
| 中性历史贝塔 | `hNeutralize(历史贝塔, log(总市值), 0)` — 仅市值(贝塔已含行业暴露) |
| 中性N日波动率 | `hNeutralize(Stdev(1日涨幅,N)*sqrt(N), log(总市值), 1)` |
| 中性N日换手率 | `hNeutralize(MA(当日换手率,N), log(总市值), 1)` |

> 说明：残差代表剔除行业/规模后的「纯」因子暴露。对数市值用于纠偏。默认申万2014分类。
> ⚠ 具体行业范围(1 vs 2)与是否含行业项，AI 未能从文档确证 —— 用前实测。

## B. 估值/财务族

| 指标 | 定义/公式 | 来源 |
|---|---|---|
| 真实负债资产率 | `负债合计 / (资产总计 - 商誉 - 无形资产 - 开发支出)` (剔除虚资产后的真实杠杆) | 平台文档(可转债系统指标列表) |
| ILLIQ | Amihud 非流动性 = `Sum(\|1日涨幅\| / 成交额(亿元), N) / N`，默认 N=5；每亿元成交额引起的股价振幅 | 平台文档(股票系统指标列表) |
| 市净率B | `总市值 / (归属母公司股东权益合计 - 其他权益工具)`，扣除优先股等，主要用于银行股；非银行股=普通市净率 | 平台文档 |
| 股息率TTM | `TTM(现金分红,0) / 价格` ≈ `过去4季现金分红 / 总市值` | 平台文档 |
| AH股溢价率 / 溢价率 | `(A股收盘价 - H股收盘价) / H股收盘价`，需 tickerValue 取 H 股价 | 平台文档 |
| 5日平均溢价率 | `MA(溢价率, 5)` | 平台文档 |
| 研发销售比率 | `研发费用(单季) / 营业收入(单季)` | ★AI 推断(文档未直接给) |
| 市研率 | `研发费用(TTM) / 总市值` | ★AI 推断 |

## C. 朝阳永续/分析师一致预期族 (数据=多机构分析师预测均值；AI 称果仁未明示是否朝阳永续)

| 指标 | 计算口径 | 来源 |
|---|---|---|
| 预期净利润1年 | 各机构预测下一年报(FY1)归母净利润的平均值 | 平台文档(股票系统指标列表) |
| 预期ROE1年 | 各机构预测 FY1 的 ROE 平均值 | 平台文档 |
| 预期EP第2年 | 预期市盈率第2年倒数 = 预期净利润第2年(FY2) / 当前总市值 | 平台文档 |
| 预期BP1年 | 预期净资产(FY1) / 当前总市值 | 平台文档 |
| 预期BP第2年 | 预期净资产(FY2) / 当前总市值 | 平台文档 |
| 预期SP1年 | 预期营业收入(FY1) / 当前总市值 | 平台文档 |
| 预期盈利2年复合增长 | 分析师预测未来2年盈利年化增长率 (B/A)^(1/2)-1 | 平台文档 |
| 预期营收2年复合增长 | 分析师预测未来2年营收年化增长率 | 平台文档 |
| 预期盈利增长率 | 分析师预测盈利增长率(口径同上复合增长族) | 平台文档 |
> 更新：随机构预测动态更新，常在财报季/业绩预告后。无机构覆盖→空值。

## D. 评级族 (数据=分析师研究报告，统计窗口 6 个月，每周更新)

| 指标 | 定义 | 来源 |
|---|---|---|
| 评级调高家数 | 过去6个月发布**调高评级**(上调目标价/评级)的机构数 | 平台文档 |
| 评级增持家数 | 过去6个月发布**增持/买入**(Buy/Outperform)评级的机构数 | 平台文档 |
| 评级机构数 | 过去6个月发布**任何评级**(买入/增持/中性/减持/卖出)的机构总数=覆盖广度 | 平台文档 |
| 分析师评级分 | 过去6个月平均评级分，1-5分(5最优) | 平台文档 |
| 近期评级变化 | `(评级调高家数 - 评级调低家数) / 评级机构总数` = 净调高比例 | 平台文档 |
| 净利润断层 | 实际净利润显著超分析师预期；超预期比率=`(实际净利-预测净利)/\|预测净利\|`，>10% 视为断层 | ★AI 推断(文档无直接定义) |

## E. 事件/风控/行业聚合族

| 指标 | 定义/公式 | 来源 |
|---|---|---|
| 果仁风险预警25版 | 退市/重大风险综合标记(1=有风险/0=无)，触发任一：①年报审计意见非标准无保留 ②年报披露逾期 ③国九条财务/交易退市(连亏/市值股价破阈) ④所有者权益为负 ⑤过去120交易日被交易所退市预警 | 平台文档 |
| 财报预约公布天数 | 距下季度财报预约公布日的剩余天数 = `当前日期 - 预约日`；未公布→空 | 平台文档 |
| 申万行业指数标记 | 股票所属申万行业代码；`industry()`=一级(如25=银行)，`industry2()`=二级 | 平台文档 |
| 行业净利润增长 | `hSum(TTM(归母净利润,0),行业) / hSum(RefQ(TTM(归母净利润,0),4),行业) - 1` (行业TTM净利同比) | ★AI 推断(hSum聚合) |
| 行业净利润环比增长 | `hSum(单季净利,行业) / hSum(RefQ(单季净利,1),行业) - 1` | ★AI 推断 |
| 行业N日涨幅 | `hAvg(N日涨幅,行业)` 或市值加权 `HWAvg(N日涨幅,总市值,行业)` | ★AI 推断 |
| 股权质押比例 | `质押股数 / 总股本`，从前10大股东公告提取，按周更新 | 平台文档(描述) |
| 管理层持股比例 | `高管合计持股数 / 总股本`，来自年报高管持股 | 平台文档(描述) |
| 10日融资偿还金额 | `Sum(当日融资偿还额, 10)`，交易所每日融资融券数据 | 平台文档(描述) |
| 未来20日新增流通股占比 | `未来20日新增流通股数 / 当前流通股数`(解禁+定增)，来自解禁计划/再融资公告 | 平台文档 |
| 过去N日重大违规数量 | 过去N日重大违规/问询函/关注函次数(CountBars 跳停牌) | 平台文档(描述) |
| 年内累计涨幅 | 当年年初至今的**后复权**收盘价累计涨幅 | 平台文档(描述) |
