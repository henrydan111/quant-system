## Four independent proofs that no decision flips

### (1) Structural — the P-GATE ceiling cannot read a residual

[replication_governance.py](../../../src/alpha_research/factor_registry/replication_governance.py))

files (`replication_governance.py`, `factor_lifecycle_steps.py`): **0 matches**. The ceiling is a
function that does not read any residual column, so the residual rebuild *cannot* move it.

### (2) Gate-input stability — the inputs it does read are residual-independent
