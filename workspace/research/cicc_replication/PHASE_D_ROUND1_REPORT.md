## 失败结构归因(全部可解释,无未知项)

1. **规模族(8 个 IC 失败格,exact)** — MC/FC/Ln_MC/Ln_FC 全市场 dIC −2.0~−3.9、
   多头 +9~+20pp:与 C 阶段 ln_mc 同根(构造无可错;中金"约2010-2022"窗口与池的
   不可观测差异 + 我方数据中微盘效应更强)。**整族标记 documented-deviation**,
   一致性本身(4 个变体同向同幅)反而支持系统性口径差而非随机错误。
2. **单调性-only 失败 25 格,其中 16 格真值单调本身 <0.6**(CSR/CUR/QR/Debt_Asset
   等弱信号比率):真值单调接近 0 时符号+幅度比对就是噪声判读。**IC 全过**
   (安全类 exact IC 12/12、盈余 3/3)——构造无问题,弱因子的形状不可对。
3. **approx 档累计口径同比族(NP_YOY/EPS_YOY/ROE_YOY 等,~12 格 IC 偏热 +1.7~+2.3pp)**:
   方向与幅度完全符合 caveat 预期(累计同比比 TTM 同比更"新鲜"→ IC 更高)。
   **近似可用作因子,但不可声称为忠实复刻。**
4. **D 后缀周转率近似失败(RATD +2.3~+3.8、ATD/INVTD csi 域)**:固定分母近似
   实质退化为营收动量,验证不通过 → **重分类为 blocked-slots**,等 D4 槽位扩展
   后用真上期分母重做。
5. AT×csi300(0.03 vs 1.93)、DPR×csi300:大盘域弱信号 + vendor 分红代理误差,单格记录。
