# 成交额集中度风控开关 — 测试结果

*Date: 2026-06-11. 来源:小红书风控建议(top-5% 成交额份额 > 均值+1.5σ → 小市值减仓至50%、
剩余转ETF)。脚本 `build_concentration.py` / `test_concentration.py` / `plot_concentration.py`。
集中度 conc[T]=当日成交额前5%个股的成交额/全市场总成交额(剔.BJ),阈值=250日trailing
均值+1.5σ(PIT,T收盘可知),信号T+1生效。ETF腿=沪深300。IS=2014+,OOS=2009-2013。*

与所有价格规则不同:它在 OOS 不毁掉收益。

## 但严格拆解后,三条都不支持把它当"小市值阴跌保护"

**① 它不降尾部。** IS MDD −48.3 ≈ 满仓 −47.8(略差)。2015 崩盘全程未保护(那年 edge −20.8pp,
减仓减在牛市里、崩盘照吃)。用户的首要诉求(降回撤)未达成。

**② 它不在你关心的崩盘上触发。** 各关键日的集中度 z 值:
- 2021-01-08 / 02-10:z=**+3.71 / +3.58** → 触发 ✓(抱团顶,教科书式正确检测)
- 2024-01-31 / 02-05(微盘流动性崩盘):z=**−1.62 / −1.35** → **不触发**
- 2025-04-07(关税崩盘):z=−0.37 → 不触发
- 2026-02-27(当前):z=+0.89 → 不触发
机制:**微盘崩盘是"广泛去杠杆"(什么都砸→换手分散→集中度低);高集中度伤的是抱团大票
(2021),不是微盘。** 信号天然瞄准的是 2021 型,不是 2024 型。它对你当前的 2026 阴跌静默。

**③ 高集中度并不预测小市值下跌,且超额大半来自分散与单一年份。**
- 小市值未来20日收益按 z 分桶(IS):z<−1 +2.32% / −1~0 +2.83% / 0~1 +3.88% / **1~1.5 +4.32%** /
  **z>1.5 +2.36%**。触发桶(z>1.5)仍是**正收益**,只是低于均值——减仓减进的是"低于平均但仍涨"
  的区间,所以减到现金是 wash(IS持平、OOS −2pp)。且最佳收益在 1~1.5(刚好触发线之下),
  阈值1.5切在了最好区间的右边。
- "转沪深300"才是超额来源:转现金版 OOS 36.7 **< 满仓38.9**(纯减仓不创造价值)。
- **2014 单年贡献约70% 的IS超额**:IS全样本 conc_switch 39.7 vs 满仓 36.9(+2.8pp);剔除2014后
  34.5 vs 33.7(**仅+0.8pp**)。2014底券商行情大票暴涨,集中度高→开关把50%押进沪深300接住melt-up。

**唯一真实的择时技巧(但不跨域稳定)**:IS 减仓日上小市值日均仅 +0.018%、沪深300 +0.143%
(轮动方向对),但 OOS 减仓日上小市值 +0.171%(高于均值,减仓是错的),靠沪深300 +0.288%
的大涨日撑住——OOS 的"超额"是ETF腿的beta,不是小市值择时。

## 与小红书原帖的差异(诚实记录)

原帖称该指标"能解释小市值今年(2024/2025)收益的很大部分"。**本复刻无法复现**:用 top-5%
定义,2024 edge −2.6pp、2025 −8.8pp——恰恰在原帖声称的年份它是负贡献,因为2024/2025微盘崩盘
是低集中度事件。差异可能来自:原帖用了不同定义(微盘内部集中度 / 大票或ETF换手份额 /
不同top-N),或对2024做了数据挖掘。若要继续,可测的替代定义:大票成交额份额、微盘内部HHI、
或集中度作为风格轮动(size factor)信号而非微盘开关。

## 判定

这是测过的最有价值的信号:维度全新(微观结构,与价格规则正交)、OOS 不毁收益、且真实捕捉到
2021 抱团拥挤(z>3.5 的正确检测)。**但它不是"小市值阴跌/回撤保护"工具**——不降尾、不在微盘
崩盘时触发、headline超额主要是大票分散+2014单年。它的真实内容是"大票换手拥挤到极端→小市值
向大票轮动",属于**风格轮动 regime 信号**,该框架下有适度价值;作为你要的下跌保护,三个验证
口径都证伪。组合 v4 能同时拿到 v4 的降尾(−25.5)与集中度的小幅提升(IS夏普1.58,全场最高),
是目前 IS 最均衡的一个,但 OOS 仍被 v4 价格腿拖到 28.7 < 满仓。
